研究方向 / Focus Areas
======

**Web AI 安全**
聚焦大语言模型（LLM）应用与 AI Agent 的安全攻防研究，涵盖 Prompt Injection、间接注入（Indirect Prompt Injection via RAG / 外部检索源）、Agent 工具滥用、RAG 知识库投毒以及 LLM 应用越狱与红队评估。关注从模型推理层到应用编排层的完整攻击链，包括跨向量数据流污染、插件 / MCP 工具安全边界以及多轮对话上下文劫持，致力于揭示 AI 驱动系统中的新型漏洞模式。

**二进制安全**
专注于漏洞挖掘与利用开发，研究方向覆盖 Fuzzing（AFL++ / libFuzzer）自动化漏洞发现、符号执行辅助路径探索（angr / Z3）、逆向分析（IDA Pro / Ghidra）以及漏洞利用构建。关注用户态与内核态漏洞利用、堆利用（tcache / fastbin / unsorted bin）、保护机制绕过（ASLR / PIE / NX / RELRO / Stack Canary）以及现代缓解下的利用原语构造。
